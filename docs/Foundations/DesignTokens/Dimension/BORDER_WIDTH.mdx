## Overview

Dimension border width tokens define the thickness of borders around various UI elements, such as buttons, containers, and dividers, by pointing to the global border-width tokens.

For more information on the code implementation of the dimension border width tokens and how to fetch them to your code [click here](../?path=/docs/utilities-theme-utilities--overview)
