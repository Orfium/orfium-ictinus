## Overview

Dimension Sizing tokens in a UI library define the dimensions of various UI elements such as typography, icons, buttons, containers, and other components, by pointing to global sizing tokens.

For more information on the code implementation of the dimension sizing tokens and how to fetch them to your code [click here](../?path=/docs/utilities-theme-utilities--overview)
