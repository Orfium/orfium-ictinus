## Updates on existing components

- All properties of components that are boolean are now e.g `loading=true` becomes `isLoading=true`

<SubsectionHeader title={'Drawer'} variant="headline03" />

- `Drawer` was renamed to `Navigation` due to its functionality.

- `expanded` changed to `isExpanded`

<SubsectionHeader title={'Overlay'} variant="headline03" />

- `Overlay` renamed to `Drawer`

- `open` changed to `isOpen`

- `size` property turned to number

<SubsectionHeader title={'List'} variant="headline03" />

- `List` is now getting children. This applies to all components using lists as well like `Menu`. Out-of-the-box items are available `ListItemText`, `ListSection`, `ListItemAction`

<SubsectionHeader title={'Menu'} variant="headline03" />

- `Menu` is now getting children. Children can be custom or items from `List`. Out-of-the-box items are available `ListItemText`, `ListSection`, `ListItemAction`

<SubsectionHeader title={'Loader'} variant="headline03" />

- `Loader` renamed to `ProgressIndicator`

- `type` changed and if you were using `dots` or `indeterminate` use `linear` and if you were using `spinner` change to `circular`

- `color` - deprecated

<SubsectionHeader title={'Pagination'} variant="headline03" />

- `Pagination` properties that changed

- `nextPageDisabled` change to `isNextPageDisabled`

- `nextPageDisabled` change to `isPrevPageDisabled`

- `hideEnhancedPaginationButtons` change to `isEnhancedPaginationVisible`

- `size` changed from `xs`, `sm`, `md`, `lg` to `1-6` see more on components tokens and globals sizing tokens
