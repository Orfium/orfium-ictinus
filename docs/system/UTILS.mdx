# System Utilities

System has some utils in place like quering, transformations etc. These utils are listed below.

---

### getColor

This util is provided from core `theme` or you can use it as a standalone function.
In case of standalone you must provide the [Palette](/?path=/story/system-theme--page#theme-palette).

It gets 3 params:

1. color
2. shade
3. scope - text, [flat](/?path=/story/guide-colors--page) or normal - default: flat
   <br />
   <br />

#### Usages <br/><br/>

##### 1) Standalone function

```
import { getColor } from 'theme';
...
getColor(palette)('lightBlue', 650)
```

##### 2) Hook

```
const theme = useTheme();
...
theme.utils.getColor('darkGrey', 650);
...
theme.utils.getColor('primary', BASE_SHADE, 'normal');
...
theme.utils.getColor('primary', BASE_SHADE, 'text');
```

---

### getColorFromType

This util provide an easy way to return the hex color from a type based on our main types 'primary', 'light', 'secondary' etc

#### Usage <br/><br/>

```
const color = getColorFromType('primary', theme)
```

---

### calculateActualColorFromComponentProp

A utility to translate a color like red-500 to an object. This calculates on the color passed picked by our palette.
So in case you run a red color for example `red-500` this will return

```
color: '#d40000'
shade: 500
```

#### Usage <br/><br/>

```
const calculatedColor = calculateActualColorFromComponentProp('#d40000');
```

---

### useTypeColorToColorMatch

A hook that on provides two methods `typesShadesColor` and `calculateColorBetweenColorAndType`

- typesShadesColor: calculates on init and returns all types like `primary` in shade and color objects
- calculateColorBetweenColorAndType: get color and type. If color exists gets it from `calculateActualColorFromComponentProp` if not and only type then it fetches it from the `typesShadesColor`

#### Usage <br/><br/>

```
const { calculateColorBetweenColorAndType } = useTypeColorToColorMatch();
const calculatedColor = calculateColorBetweenColorAndType(color, type);

console.log(calculatedColor.color, calculatedColor.shade);
```

### getHover

A util that returns the global state for hovered components.

It gets 3 params:

1. theme,
2. color ( default lightGrey ),
3. shade ( default 0)

Returns:

```
{
    backgroundColor: string
}
```

#### Usage <br/><br/>

```
background: getHover({theme, color: 'blue', shade: 500}).backgroundColor;
background: getHover({theme, color: 'blue'}).backgroundColor;
background: getHover({theme}).backgroundColor;
```

---

### getFocus

A util that returns the global state for focused components.

It gets 2 params:

1. theme,
2. borderWidth ( default 0 ),

Returns:

```
{
  borderWidth: string;
  focusColor: string;
  styleBorder: string;
  styleOutline: string;
}
```

#### Usage <br/><br/>

```
border: `${getFocus({theme, borderWidth: 2 }).borderWidth solid getFocus({theme, borderWidth: 2 }).focusColor}`;
border: getFocus({theme}).styleBorder;
outline: getFocus({theme}).styleOutline;
```

---

---

### getDisabled

A util that returns the global state for disbaled components.

It gets 0 params and returns:

```
{
  opacity: number;
  cursor: string;
}
```

#### Usage <br/><br/>

```
cursor: getDisabled().cursor;
opacity: getDisabled().opacity;
```
