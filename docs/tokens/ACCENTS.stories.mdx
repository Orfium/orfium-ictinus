## Overview

Accent colors are used for **non-interactive components** such as tags, avatars, or other decorative elements. They are designed to provide visual interest and add personality to the UI.

| Type         | Description                                                                                              |
| ------------ | -------------------------------------------------------------------------------------------------------- |
| **main**     | Main color of component, default color.                                                                  |
| **contrast** | Used only on `accents` type for AA compiance for small text when `light` or `main` values of same color. |
