## Overview

Systemic colors are the heart ❤️of the design system. Systemic colors are a set of predefined colors using colors from the main [palette colors](?path=/docs/guide-colors--page), used consistently across **interactive components** in a design system.

<br />
Each token have specific structure of `light`, `main`, and `dark` color. Each type usually is used to
define a specific state

| Type      | Description                                             |
| --------- | ------------------------------------------------------- |
| **light** | Used on the hover state of the component                |
| **main**  | Main state of component, default color.                 |
| **dark**  | Used when the components is under focus or press state. |

<br />
<Tip>
  Most of the times you will need colors from these but if you need to use colors for a
  non-interactive component, check out{' '}
  <LinkTo kind="/Design Tokens/Semantic/Colors/Accent Colors">Accent Colors</LinkTo>
</Tip>
