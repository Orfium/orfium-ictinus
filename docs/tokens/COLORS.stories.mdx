## Overview

Systemic colors are tokens that holds colors from the main [palette colors](?path=/docs/guide-colors--page). Each token have specific structure of `light`, `main`, and `dark` color.
Each type usually is used to define a specific state

| Type      | Description                                             |
| --------- | ------------------------------------------------------- |
| **light** | Used on the hover state of the component                |
| **main**  | Main state of component, default color.                 |
| **dark**  | Used when the components is under focus or press state. |
