## Overview

Proper sizing helps create visual hierarchy within a page and directs the user towards a desired action. Spacing in general uses multipliers of 4 for a consistent visual effect (with the exception of spacing.0). All designs should use exclusively the sizing tokens provided below.

## Tokens

Sizing is part of [globals](?path=/docs/design-tokens-about--page#globals) tier of tokens.
