## Overview

Palette colors are a set of predefined colors using colors from the [global colors](?path=/docs/design-tokens-global-colors--page), used consistently across **interactive components** in a design system.

<br />
Each token have specific structure of lightest, light, main, dark and darkest color. Each type usually
is used to define a specific state.
