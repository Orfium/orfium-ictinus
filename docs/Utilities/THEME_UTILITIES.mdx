## Get Functions

In order to retrieve styles from the library's tiers (global, theme, dimension), some specific `get` utils can be used.

<SubsectionHeader title={'Standalone Function'} variant={'headline04'} />

```
import { themeConfig } from '@orfium/ictinus'

...

themeConfig('semantic').globals.borderWidth.get('1');
// returns '0.0625rem'

themeConfig('semantic').tokens.colors.get('backdrop.default');
// returns 'rgba(200,206,255,0.70)'

```

On the example above, the theme (`semantic`) is specified as a parameter.

<SubsectionHeader title={'Hook'} variant={'headline04'} />

```
import { useTheme } from '@orfium/ictinus'

...

const theme = useTheme();

theme.globals.borderWidth.get('1');
// returns '0.0625rem'

theme.tokens.colors.get('backdrop.default');
// returns 'rgba(200,206,255,0.70)'

```

#### Theme (Semantic)

<div className="documentation-table">

| Property       | Get function                     |
| -------------- | -------------------------------- |
| Colors         | theme.tokens.colors.get()        |
| Disabled State | theme.tokens.disabledState.get() |
| Icon           | theme.tokens.icon.get()          |
| State          | theme.tokens.state.get()         |
| Typography     | theme.tokens.typography.get()    |
| Box Shadow     | theme.tokens.boxShadow.get()     |

</div>

For `Typography` the return value is an object with css properties as keys and their css values as values.

```
import { useTheme } from '@orfium/ictinus'

...

const theme = useTheme();

theme.tokens.typography.get('normal.body04');

/**
 * returns
 *  {
      "fontFamily": "Roboto",
      "fontWeight": 400,
      "lineHeight": "0.875rem",
      "fontSize": "0.625rem",
      "letterSpacing": "0.00625rem"
    }
 *
 */

```
