### Sticky Header

In order to create a table with a sticky header:

1. Add the `hasStickyHeader` prop
2. Add a `max-height` to the tBody as indicated below:

```js
<Table hasStickyHeader sx={{ tbody: { maxHeight: '300px' } }} ...rest />
```
