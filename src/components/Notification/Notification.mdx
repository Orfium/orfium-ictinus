## Toast Notification

### Toast Props

<ArgTypes of={Toast} />

### NotificationVisual Props

<ArgTypes of={NotificationVisual} />

### Usage

```js
import { Toast, NotificationVisual } from '@orfium/ictinus';

<NotificationsContainer position="bottom-right">
  <Toast message="message" type="info" isExpanded hasMinimumHeight={false} closeCTA={closeCTA}>
    <NotificationVisual
      title="title"
      primaryCTALabel="Primary CTA Label"
      primaryCTA={primaryCTA}
      secondaryCTALabel="Secondary CTA Label"
      secondaryCTA={secondaryCTA}
      description="Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed viverra neque nec est porttitor"
    />
  </Toast>
</NotificationsContainer>;
```
