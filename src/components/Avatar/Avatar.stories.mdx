# Avatar

A universal Avatar component cam be found around ictinus on menus or can be a standalone component as well. It renders either images, letter or icon from our library

# Usage

```js
import { Avatar } from '@orfium/ictinus';

<Avatar src={'https://mui.com/static/images/avatar/1.jpg'}>PV</Avatar>;
```

# Avatar Stack

A universal AvatarStack component that renders its children as a stack.

# Usage

```js
import { Avatar, AvatarStack } from '@orfium/ictinus';

<AvatarStack>
  <Avatar/>
  <Avatar/>
  <Avatar/>
</AvatarStack>
```
