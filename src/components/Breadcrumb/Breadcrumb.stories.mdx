# Breadcrumb

A Breadcrumb component that can be configured either by adding children components (Usage 1) or bypassing the data property (Usage 2).
It contains a collapsed view. In case components overcome the maximum value of Breadcrumb items which is hardcoded to 4.

# Usage 1

```js
import { Breadcrumb } from '@orfium/ictinus';

<Breadcrumb>
  <div>content 1</div>
  <div>content 2</div>
  <div>content 3</div>
</Breadcrumb>;
```

# Usage 2

```js
import { Breadcrumb } from '@orfium/ictinus';

<Breadcrumb data={[{ to: '/path', label: 'label' }]} />;
```
