## Overview

<Overview>
  <Typography style={{ paddingBottom: '16px' }}>
    The Box component is a helper component that lets you write these common less-reusable

  </Typography>
  <Typography style={{ paddingBottom: '16px' }}>
    One of the most common needs when building web apps is creating layout elements that have
    certain layout.
  </Typography>
  <Typography style={{ paddingBottom: '16px' }}>
    The mentality is that we use all CSS props based on{' '}

    .
  </Typography>

    t|r|b|l ending with m + t | r | l | b = value
  </Typography>
</Overview>
