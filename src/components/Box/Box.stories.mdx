### Box Horizontal Padding

Padding is the same as "margin" in the application, you can also use `mx` for margin.

<Preview>
  <Story name="Box Horizontal Padding">
    <Box backgroundColor={'default'} px={'7'}>
      Padding Horizontal 7
    </Box>
  </Story>
</Preview>

### Box Vertical Padding

Padding is the same as "margin" in the application, you can also use `my` for margin.

<Preview>
  <Story name="Box Vertical Padding">

    </Box>
  </Story>
</Preview>
