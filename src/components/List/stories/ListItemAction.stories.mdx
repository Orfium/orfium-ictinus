## Overview

The ListItemAction component represents an action or a set of actions associated with a ListItem. It can contain buttons, icons, or any other interactive elements.

## Usage

<UsageGuidelines guidelines={['use only for buttons, icons, or any other interactive elements']} />

```js
import { List, ListItem, ListItemAction, Radio, Checkbox } from '@orfium/ictinus';

const MyComponent = () => {
  return (
    <List>
      <ListItem>
        <ListItemAction>
          <Radio isChecked={false} isDisabled={false} />
        </ListItemAction>
      </ListItem>
      <ListItem>
        <ListItemAction>
          <Checkbox isChecked={false} isDisabled={false} />
        </ListItemAction>
      </ListItem>
    </List>
  );
};
```
