## Overview

<Overview>
  The ListItemText component is used within a ListItem to display textual content with a primary and
  a secondary text.
</Overview>

## Usage

<UsageGuidelines
  guidelines={['Always use to show text without other custom components like Icon, Button etc']}
/>

```js
import { List, ListItem, ListItemText } from '@orfium/ictinus';

const MyComponent = () => {
  return (
    <List>
      <ListItem>
        <ListItemText>Item 1</ListItemText>
      </ListItem>
      <ListItem>
        <ListItemText description="This is a secondary text on this item">Item 2</ListItemText>
      </ListItem>
    </List>
  );
};
```
