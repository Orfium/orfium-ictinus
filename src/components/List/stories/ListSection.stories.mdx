## Overview

The ListSection component is used to group related items within the List.

## Usage

<UsageGuidelines
  guidelines={[
    'It can render ONLY individual ListItem components',
    'It can contain multiple ListItem components as its children',
    'Use ListSection when you have a collection of items that share a common category or relationship.',
  ]}
/>

```js
import { List, ListSection, ListItem } from '@orfium/ictinus';

const MyComponent = () => {
  return (
    <List>
      <ListSection title="Section 1">
        <ListItem>...</ListItem>
        <ListItem>...</ListItem>
      </ListSection>

      <ListSection title="Section 2">
        <ListItem>...</ListItem>
        <ListItem>...</ListItem>
      </ListSection>
    </List>
  );
};
```
