A universal IconButton component that with the help of Button hold the icon inside it and makes it clickable and easily accessible.

## Usage

```js
import { IconButton } from '@orfium/ictinus';

<IconButton onClick={() => {}} name={'add'} />;
```

## Props

<Props of={IconButton} />

# IconButton Types and Sizes

IconButton with different types `primary, secondary, tertiary` and different shape types `circle, square` <br/>
There are 2 IconButton sizes, `normal` and `compact`.
