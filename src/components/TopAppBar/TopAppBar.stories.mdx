# TopAppBar

A universal TopAppBar component.

## Usage

```js
import { TopNavBar } from '@orfium/ictinus';

<TopNavBar
  isDark={false}
  logoIcon={<Icon />}
  onMenuIconClick={() => {}}
  userMenu={{
    items: ['item1', 'item2'],
    userAvatar: { letter: 'JD', src: 'http://lorem.com/john-doe-profile.jpg' },
    userName: 'John Doe',
    onSelect: () => {},
  }}
  additionalTools={<div>Extra Information</div>}
  onSearchHandler={() => {}}
/>;
```
