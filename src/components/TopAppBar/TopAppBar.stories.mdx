# TopAppBar

A universal TopAppBar component.

# Usage

```js
import { TopAppBar } from '@orfium/ictinus';

<TopAppBar
     dark={dark}
     logoIcon={logoIcon}
     onMenuIconClick={onMenuIconClick}
     userMenu={userMenu}
     additionalTools={<AdditionalToolsComponent />}
     onSearchHandler={onSearchHandler}
/>
```
