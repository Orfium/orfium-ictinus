## Overview

This component can be rendered as a child to the Drawer component and creates a header for the Drawer with a close button for the Drawer included.

## Props

<ArgTypes of={DrawerHeader} />

## Usage

This component uses Drawer's context to receive the `hasFixedLayout` boolean state and the `onClose` callback provided to Drawer as a prop, in order to create the header functionality expected.

```
import Drawer, { DrawerHeader } from '@orfium/ictinus';

const MyComponent = () => {
  return (
    <Drawer isOpen={true} hasFixedLayout={true} onClose={() => console.log('close')} size={33} anchor={'left'}>
      <DrawerHeader>
        <div>This is the header</div>
      </DrawerHeader>
    </Drawer>
  );
};

```

In order to use the drawer context state, import the `useDrawerContext` hook.

```
import { useDrawerContext } from '@orfium/ictinus';

const CustomHeader = () => {
  const { hasFixedLayout, onClose } = useDrawerContext();

  return <div>Custom Header</div>;
};

```
