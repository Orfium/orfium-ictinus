## Overview

This component can be rendered as a child to Drawer component and creates a body content for the Drawer.

## Props

<ArgTypes of={DrawerContent} />

## Usage

This component creates a body content for the Drawer. If the `hasFixedLayout` prop is provided to the Drawer component, then this component becomes scrollable.

```
import Drawer, { DrawerContent } from '@orfium/ictinus';

const MyComponent = () => {
  return (
    <Drawer isOpen={true} onClose={() => console.log('close')} size={33} anchor={'left'}>
      <DrawerContent>
        <div>This is the drawer's content</div>
      </DrawerContent>
    </Drawer>
  );
};

```
