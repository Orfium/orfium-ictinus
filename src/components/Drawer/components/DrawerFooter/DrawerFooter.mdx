## Overview

This component can be rendered as a child to Drawer component and creates a footer for the Drawer.

## Props

<ArgTypes of={DrawerFooter} />

## Usage

When DrawerFooter is used, it creates a fixed footer inside the Drawer container. This component uses Drawer's context to receive the `hasFixedLayout` boolean state, in order to create the footer functionality expected.

```
import Drawer, { DrawerFooter } from '@orfium/ictinus';

const MyComponent = () => {
  return (
    <Drawer isOpen={true} hasFixedLayout={true} onClose={() => console.log('close')} size={33} anchor={'left'}>
      <DrawerFooter>
        <div>This is the footer content</div>
      </DrawerFooter>
    </Drawer>
  );
};

```

In order to use the drawer context state, import the `useDrawerContext` hook.

```
import { useDrawerContext } from '@orfium/ictinus';

const CustomFooter = () => {
  const { hasFixedLayout, onClose } = useDrawerContext();

  return <div>Custom Footer</div>;
};

```
