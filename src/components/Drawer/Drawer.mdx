# Drawer

A universal Drawer component

## Usage

```js
import { Drawer } from '@orfium/ictinus';

<Drawer expanded={expanded} menuItems={menuItems} />;
```

<b>Important tip: </b>

When rendering the Drawer component, the parent component should have `position: relative` in order for the Drawer to align correcly since it has `position: absolute`.
