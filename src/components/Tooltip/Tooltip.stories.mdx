# Tooltip

# Usage

```js
import { Tooltip } from '@orfium/ictinus';

<Tooltip content="Content you want to show as descrption of tooltip" placement={'right'}>
  ...
</Tooltip>;
```

# Generic Tooltip

Tooltip use to show info in any direction defined. If there is no space in the screen then it moves to the available space.
