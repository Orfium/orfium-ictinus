# Tooltip

Generic tooltip that can show content as string
The background defined is set as property to ThemeProvider with `overrides` property because color for this is global and cannot be changed per case.
e.g `overrides: { tooltip: { background: { color: 'darkGray', shade: '500' } } }`

##

```js
import { Tooltip } from '@orfium/ictinus';

<Tooltip content="Content you want to show as descrption of tooltip" placement={'right'}>
  ...
</Tooltip>;
```

<Props of={Tooltip} />

### Generic Tooltip

Tooltip use to show info in any direction defined. If there is no space in the screen then it moves to the available space.
