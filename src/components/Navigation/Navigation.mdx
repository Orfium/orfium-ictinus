# Navigation

A universal Navigation component

## Usage

```js
import { Navigation } from '@orfium/ictinus';

<Navigation expanded={expanded} menuItems={menuItems} />;
```

<b>Important tip: </b>

When rendering the Navigation component, the parent component should have `position: relative` in order for the Navigation to align correcly since it has `position: absolute`.
