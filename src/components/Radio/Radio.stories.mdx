# Radio

A custom made radio input based on the Orfium design system. Passing a `ref` will return the actual input element.
If the `checked` prop is not defined it means the `Radio` is not controlled.

##

```jsx
import { Radio } from '@orfium/ictinus';

<Radio isChecked={value === 'a'} value={'a'} onChange={() => {}} />;
```
