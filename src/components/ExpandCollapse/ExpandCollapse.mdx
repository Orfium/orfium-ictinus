## Overview

A minimally visual component that allows to expand and collapse content

## Props

<ArgTypes of={ExpandCollapse} />

## Usage

```jsx
import { ExpandCollapse } from '@orfium/ictinus';

<ExpandCollapse
  textAndControl={(handleClick) => {
    return (
      <Fragment>
        <span>Text</span>
        <button onClick={handleClick}>expand/collapse</button>
      </Fragment>
    );
  }}
>
  {(expand) => {
    return (
      <div>
        <div>Some</div>
        <div>content</div>
        <div>here</div>
      </div>
    );
  }}
</ExpandCollapse>;
```
