## Overview

TabList is the wrapper of the Tab components. Here you can use the sx prop to add custom styles for the Tabs.

## Props

<ArgTypes of={TabList} />

## Usage

Use the TabList to add aria-label and style overrides.

<Tip>
  Check also
  [TabsContainer](../?path=/docs/updated-components-tabs-components-tabscontainer--overview) and
  [Tab](../?path=/docs/updated-components-tabs-components-tab--overview) as they are needed for the
  implementation.
</Tip>

<Tip>
  When using custom tab components for children, use `role='tab'` on every child tab in order for
  the TabList styles to be applied.
</Tip>
