# DropdownButton

A Button with a CTA and a dropdown with actions

# Usage

```js
import { DropdownButton } from '@orfium/ictinus';

<DropdownButton
  items={['Item 1', 'Item 2']}
  onButtonClick={() => console.log('click')}
  onOptionSelect={(option) => console.log(option)}
>
  Click Here
</DropdownButton>;
```
