# TopNavBar

A universal TopNavBar component.

# Usage

```js
import { TopNavBar } from '@orfium/ictinus';

<TopNavBar
     dark={dark}
     logoIcon={logoIcon}
     onMenuIconClick={onMenuIconClick}
     userMenu={userMenu}
     additionalTools={<AdditionalToolsComponent />}
     onSearchHandler={onSearchHandler}
/>
```
