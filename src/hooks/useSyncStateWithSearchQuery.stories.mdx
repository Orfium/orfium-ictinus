# useSyncStateWithSearchQuery

The useSyncStateWithSearchQuery hook, syncs the URL search query parameters with a component state.

Currently we support (as state) any key - value plain object that is a **Record<string, string | number>**. Also, please keep in mind that this hook can only be used into a react-router-dom context.

# Usage

```js
import { useSyncStateWithSearchQuery } from '@orfium/ictinus/dist/hooks';

const MyComponent: FC = () => {
  // URL => https://orfium-product.com/mypath?page=1&size=12&library=react&testing=yolo
  const searchQueryParams = useSearchQueryParams();

  // console.log(searchQueryParams)
  // { "page": "1", "size": "12", "library": "react", "testing": "yolo" }
};
```
